# Weather Diary - Дневник погоды

## Автор
Черникова Кристина

## Описание программы
Десктоп-приложение для ведения дневника погоды с возможностью добавления, редактирования, фильтрации и анализа погодных записей. Данные сохраняются в JSON файл.

### Возможности:
- 🌤️ Добавление записей о погоде (дата, температура, описание, осадки)
- 🔍 Фильтрация по дате и температуре
- ✏️ Редактирование и удаление записей
- 📊 Подробная статистика и анализ погоды
- 💾 Автоматическое сохранение в JSON
- ✅ Валидация всех полей ввода
